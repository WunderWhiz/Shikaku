Shikaku Web (Free Hosted)

Files:
- index.html
- style.css
- app.js

How to run locally:
- Open index.html in a browser.

How to host free (GitHub Pages):
1) Create a GitHub repo (example: shikaku)
2) Upload these files to the repo root
3) Repo Settings -> Pages -> Deploy from branch (main) -> /(root)
4) Visit the published URL shown by GitHub Pages

Controls:
- Drag to draw rectangles
- Only rectangles with exactly one clue number and matching area will place
- Tap any filled cell to delete that rectangle
- Timer starts on first placement and stops when solved
- Next Puzzle generates a new random puzzle for the selected size
